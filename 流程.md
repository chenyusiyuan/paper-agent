V1 完整启动与测试流程                                                                           
                                                                                                  
  第 0 步：环境准备（一次性）                                                                     
                                                                                                  
  # 安装依赖                                                                                      
  pip install -r requirements.txt                                                               
                                                                                                  
  # 配置环境变量（复制并填写）                                                                    
  cp .env.example .env                                                                            
                                                                                                  
  .env 最少需要填写：                                                                             
  LLM_API_KEY=your_key                                                                            
  LLM_BASE_URL=https://api.deepseek.com/v1                                                        
  LLM_MODEL=deepseek-chat

  ---
  第 1 步：运行单元测试（无需任何服务，随时可跑）

  # 在项目根目录
  PYTHONPATH=. pytest tests/test_agent tests/test_retrieval tests/test_parsing tests/test_indexing tests/test_evaluation tests/test_generation -v

  跳过需要 TestClient 的 API 测试（环境已知挂住）：
  PYTHONPATH=. pytest tests/ --ignore=tests/test_api -v

  预期：全部绿。

  ---
  第 2 步：启动 GROBID

  docker compose up -d grobid

  # 等待就绪（约 30 秒）
  curl http://localhost:8070/api/isalive
  # 返回 true 表示就绪

  ---
  第 3 步：入库论文（解析 + 建索引）

  把 PDF 放入 data/raw_pdfs/，然后：

  # 方式一：一步完成（解析 + 建索引）
  PYTHONPATH=. python scripts/ingest_papers.py --pdf-dir data/raw_pdfs

  # 方式二：已有 parsed JSON 时只重建索引
  PYTHONPATH=. python scripts/build_indexes.py

  产物写入：
  - data/parsed/*.json — 解析结果
  - data/indexes/ — FAISS + BM25 + metadata pickle

  ---
  第 4 步：启动 API 服务

  PYTHONPATH=. uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

  快速验证：
  # 列出已入库论文
  curl http://localhost:8000/api/papers

  # 发起一次对话
  curl -X POST http://localhost:8000/api/chat \
    -H "Content-Type: application/json" \
    -d '{"query": "有什么关于RAG的论文", "session_id": "test-1"}'

  ---
  第 5 步：启动 Gradio Demo

  PYTHONPATH=. python app/demo.py
  # 默认访问 http://127.0.0.1:7860

  Demo 功能：
  - 左侧聊天框对话，右侧实时展示 AgentState + 候选论文/证据
  - Lock Paper 输入框手动锁定论文（替代"选第一篇"自然语言确认）

  ---
  第 6 步：构建评测集 & 跑评测

  # 交互式录入评测样本（需要索引已建好）
  PYTHONPATH=. python scripts/build_evalset.py
  # 按提示逐条输入 question / intent_type / gold_paper_ids / gold_answer
  # 结果保存至 data/eval/evalset.jsonl

  # 运行评测（需要 LLM API 可用）
  PYTHONPATH=. python scripts/run_eval.py
  # 输出：意图路由准确率 + LLM-as-a-Judge 分数 + A/B 对比
  # CSV 结果保存至 data/eval/results.csv

  ---
  流程总览

  安装依赖 → 配置 .env
        ↓
  pytest（无服务依赖，随时验证代码）
        ↓
  docker compose up grobid
        ↓
  ingest_papers.py（PDF → parsed JSON → indexes）
        ↓
  uvicorn（API 服务）  ←→  demo.py（Gradio）
        ↓
  build_evalset.py（手工标注）→ run_eval.py（出评测报告）
